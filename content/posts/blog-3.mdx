## react-helmet

**react-helmet** 을 이용하면 컴포넌트내에서 head 태그를 수정할 수 있게 됩니다.

1. react-helmet 설치

```bash
npm install react-helmet
```

2. Helmet 태그를 활용한 컴포넌트 작성

```jsx
import React from "react";
import Icon from "../images/notebook.png";
import { Helmet } from "react-helmet";

export default function Head() {
  return (
    <Helmet>
      <title>hyeonzii의 블로그</title>
      <meta
        name="description"
        content="프론트엔드 개발자 이현지의 기술 블로그 입니다!"
      />
      <link rel="shortcut icon" href="favicon.ico" />
      <link rel="icon" href={Icon} />
      <link rel="icon" type="image/png" href="favicon.ico" sizes="16x16" />
    </Helmet>
  );
}
```

저는 title이랑 description 도 정의해주었습니다.
